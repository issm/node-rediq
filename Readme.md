# CONFIGURATION

## Connection to Redis

You can configure using **environment variables**.

### `REDIQ_REDIS_HOST`, `REDIS_HOST = "localhost"`

### `REDIQ_REDIS_PORT`, `REDIS_PORT = "6379"`

### `REDIQ_REDIS_DB`, `REDIS_DB = "1"`
